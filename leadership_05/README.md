# 중간계 리더십 실전코스 웹페이지

## 개요
이 프로젝트는 중간계 리더십 실전코스를 위한 반응형 웹페이지입니다. 웹과 모바일 환경에서 모두 최적화되어 있으며, 다양한 리더십 관련 기능을 제공합니다.

## 주요 기능
- **나의 리더십 유형 테스트**: 12개 문항을 통해 자신의 리더십 유형을 파악할 수 있습니다.
- **직원 스타일 가이드**: 다양한 직원 유형과 그에 맞는 대처법을 제공합니다.
- **웃음 채점기**: 카메라를 통해 웃음을 측정하고 점수를 제공합니다.
- **리더십 챗봇**: 리더십 관련 질문에 답변해주는 대화형 챗봇입니다.

## 기술 스택
- HTML5
- CSS3 (반응형 디자인)
- JavaScript (ES6+)
- 웹캠 API

## 실행 방법
1. 저장소를 클론합니다.
```
git clone [repository-url]
```

2. 프로젝트 폴더로 이동합니다.
```
cd leadership_05
```

3. 웹 서버를 통해 index.html 파일을 실행합니다. 간단한 방법으로는 VS Code의 Live Server 확장 프로그램을 사용하거나, Python의 내장 HTTP 서버를 사용할 수 있습니다.
```
# Python을 사용하는 경우
python -m http.server
```

4. 웹 브라우저에서 `http://localhost:8000`(또는 설정된 포트)으로 접속합니다.

## 반응형 디자인
- PC 환경에서는 풀스크린 레이아웃으로 최적화되어 있습니다.
- 모바일 환경에서는 햄버거 메뉴를 통해 내비게이션에 접근할 수 있습니다.
- 모든 컴포넌트는 화면 크기에 따라 자동으로 조정됩니다.

## 파일 구조
- `index.html`: 메인 HTML 파일
- `styles.css`: 스타일시트
- `script.js`: JavaScript 로직
- `logo.svg`: 로고 이미지

## 주의사항
- 웃음 채점기 기능을 사용하려면 카메라 접근 권한을 허용해야 합니다.
- 최신 브라우저에서 최적화되어 있습니다 (Chrome, Firefox, Safari, Edge 지원).
